## 2.0版本  
###### update: 2022-3-23

## 功能实现：
#### 玩家可选择不同难度开始游戏
#### 英雄机发射子弹使敌机坠落，敌机发射子弹，坠毁后随机掉落道具
#### 普通敌机不发射子弹，精英敌机直射子弹，boss敌机散射子弹
#### 加血道具可为英雄机恢复一定血量，炸弹道具可销毁除boss外的敌机及其子弹，火力道具可以增加英雄机火力
#### 玩家可选择是否播放音乐
#### 游戏结束后显示对应难度的排行榜，玩家可选择删除游戏记录
